# Phase 2: DILI Generalization Scripts

Scripts for scoring compounds at scale and computing ROC/AUC.

## Scripts

- `score_all_compounds.py` - Run ECNP on all curated compounds
- `compute_roc_auc.py` - ROC curve and AUC with bootstrap CI
- `stratified_analysis.py` - AUC by target count, drug class, severity
- `calibration_analysis.py` - p-value calibration curves
