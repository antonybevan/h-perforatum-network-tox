# Phase 4: Method Benchmarking

Scripts for comparing ECNP against competing methods.

## Scripts

- `implement_proximity.py` - Network proximity baseline (Guney 2016)
- `implement_naive.py` - Naive target count baseline
- `run_benchmark.py` - Head-to-head comparison
- `ablation_study.py` - ECNP component contribution analysis
- `statistical_tests.py` - DeLong test for AUC comparison
