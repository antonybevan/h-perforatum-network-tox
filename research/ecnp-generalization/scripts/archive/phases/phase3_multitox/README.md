# Phase 3: Multi-Toxicity Extension

Scripts for building tissue-specific networks and cross-toxicity validation.

## Scripts

- `build_heart_network.py` - Cardiotoxicity network (GTEx heart)
- `build_kidney_network.py` - Nephrotoxicity network (GTEx kidney)
- `build_brain_network.py` - Neurotoxicity network (GTEx brain)
- `cross_toxicity_analysis.py` - Compare rankings across endpoints
