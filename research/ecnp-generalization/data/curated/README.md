# Curated Data

This folder will contain curated compound-target and toxicity label databases.

## Structure

```
curated/
├── compound_targets/
│   ├── drugbank_targets.csv      # DrugBank compound-target pairs
│   └── chembl_targets.csv        # ChEMBL bioactivity data
└── toxicity_labels/
    ├── dili_labels.csv           # DILIrank DILI labels
    ├── cardiotox_labels.csv      # CredibleMeds cardiotoxicity
    └── nephrotox_labels.csv      # Nephrotoxicity labels
```

## Data will be populated by Phase 1 scripts.
