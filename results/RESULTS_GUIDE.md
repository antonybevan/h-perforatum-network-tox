## Core Result Files

### 1. `final_statistics.csv` ⭐ (Primary Results)

**RWR influence analysis at STRING ≥900 threshold**

| Column | Description |
|--------|-------------|
| `compound` | Drug compound name |
| `n_targets` | Number of targets mapped to LCC |
| `influence_total` | Sum of RWR scores reaching DILI genes |
| `influence_per_target` | Total influence ÷ target count |
| `null_mean` | Expected influence (random) |
| `null_std` | Standard deviation of null |
| `z_score` | (observed - null_mean) / null_std |
| `p_value` | One-tailed significance |
| `p_corrected` | FDR-corrected p-value |
| `significant` | True if p_corrected < 0.05 |

### 2. `final_statistics_700.csv` (Robustness Check)

Same format as above, but using STRING ≥700 threshold network.

| Compound | Z-score (≥900) | Z-score (≥700) | Consistent? |
|----------|----------------|----------------|-------------|
| Hyperforin | +9.50 | +6.49 | ✅ Both significant |
| Quercetin | +1.04 | +0.98 | ✅ Both NS |

### 3. `bootstrap_sensitivity.csv`

**Validates that results are robust to target count differences**

| Column | Description |
|--------|-------------|
| `bootstrap_iteration` | Iteration number (1-1000) |
| `sampled_targets` | Number of randomly sampled targets |
| `influence_per_target` | Influence for that sample |

**Key metrics:**
- Quercetin 95% CI: [0.0035, 0.0877]
- Hyperforin observed: 0.2579 (outside Quercetin CI)

## Interpretation Guide

### Z-Score Thresholds

| Z-Score | Significance | P-value |
|---------|--------------|---------|
| > 1.96 | Significant | < 0.05 |
| > 2.58 | Highly significant | < 0.01 |
| > 3.29 | Very highly significant | < 0.001 |
| > 4.00 | Extremely significant | < 0.0001 |

### Metric Interpretation

| Metric | Higher = | Lower = |
|--------|----------|---------|
| RWR Z-score | More DILI influence | Less DILI influence |
| Shortest path Z | Farther from DILI | Closer to DILI |
| Per-target influence | More potent | Less potent |
