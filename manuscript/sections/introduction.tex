\section{Introduction}

Network-based toxicology assumes that compounds targeting many proteins or lying close to disease-associated genes pose greater risk.
This intuition conflates topological proximity with functional influence and lacks a principled way to distinguish diffuse polypharmacology from high-leverage perturbation.
The assumption is testable.

\textit{Hypericum perforatum} (St.\ John's Wort) provides a controlled test case.
The extract contains two major bioactive classes with contrasting target profiles: the phloroglucinol Hyperforin (few targets, strong PXR activation) and the flavonoid Quercetin (many targets, distributed pharmacology) \citep{Nahrstedt1997, Moore2000}.
Both have hepatic activity.
Which constituent drives hepatotoxicity risk remains unresolved \citep{Hammerness2003, EMA2018}.

We construct a liver-expressed human protein--protein interaction network and systematically separate descriptive proximity from inferential influence using random walk--based propagation \citep{Kohler2008, Guney2016}.
To resolve the target-count paradox---where compounds with more targets may exert weaker influence---we introduce per-target network influence (PTNI), which normalizes propagated influence by target set size.
PTNI reframes polypharmacology as an efficiency problem rather than a coverage problem.

We compare Hyperforin (10 targets) and Quercetin (62 targets) across proximity, influence, expression-weighted influence, PTNI, bootstrap sensitivity, and chemical similarity.
The analysis tests whether target count and proximity predict risk, or whether influence efficiency provides a more robust criterion.
