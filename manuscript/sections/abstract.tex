\section*{Abstract}

Network-based toxicology uses proximity to disease-associated genes as an implicit prioritization criterion, assuming that compounds with more targets or closer network position pose greater risk.
Proximity is a descriptive metric, yet its use for prioritization conflates topological reachability with functional influence and provides no principled criterion for distinguishing diffuse polypharmacology from high-leverage perturbation.

We test this assumption using two \textit{Hypericum perforatum} constituents---Hyperforin (10 targets) and Quercetin (62 targets)---as a controlled comparison for hepatotoxicity risk.
On a liver-expressed human protein--protein interaction network, we separate descriptive proximity from inferential influence using random walk--based propagation.
Although Quercetin exhibits closer shortest-path proximity to DILI genes at STRING confidence $\geq$900, Hyperforin exerts 2.3-fold greater network influence (Z = +10.3 vs.\ +4.4; $p < 10^{-16}$).
This ranking persists under expression-weighted constraints.

To resolve the target-count paradox, we apply per-target network influence (PTNI), a normalization that quantifies influence efficiency rather than raw coverage.
Hyperforin achieves 22-fold higher PTNI than Quercetin.
Bootstrap resampling (Hyperforin exceeds 95\% CI of target-matched Quercetin subsets; 3.7$\times$ fold over mean) and chemical similarity analysis (neither compound resembles known hepatotoxins) exclude target-count and structural confounding.

These results demonstrate that proximity provides useful descriptive context but yields threshold-dependent rankings that are insufficient for inferential prioritization; influence-based metrics are more robust.
The framework---combining influence propagation, expression weighting, PTNI normalization, and orthogonal controls---generalizes to any polypharmacological comparison requiring constituent-level risk attribution.

\noindent\textbf{Keywords:} per-target network influence, drug-induced liver injury, network pharmacology, random walk with restart, \textit{Hypericum perforatum}, Hyperforin, polypharmacology
