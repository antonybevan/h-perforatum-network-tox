\section*{Abstract}

Network proximity is widely used to contextualize relationships between compounds and disease-associated genes, yet it is often interpreted as a prioritization signal despite its sensitivity to target set size, network density, and null variance.
This conflation of descriptive proximity with functional influence complicates risk assessment in polypharmacological systems.

Here, we systematically separate descriptive and inferential roles of network metrics using a liver-expressed human protein--protein interaction network.
Shortest-path proximity is evaluated as contextual information, while functional influence is quantified via random walk--based propagation and assessed against degree-matched null models.
Using two constituents of \textit{Hypericum perforatum} as a controlled stress test, we show that proximity-based rankings are unstable across interaction confidence thresholds and null specifications, whereas influence-based rankings remain consistent.

Although quercetin targets more proteins and exhibits greater proximity surprise relative to its compound-specific null, it exerts substantially weaker propagated influence than hyperforin across both topology-only and expression-weighted analyses.
To account for differences in target set size, we apply a per-target normalization of propagated influence, reframing polypharmacology as an efficiency problem rather than a coverage problem.
This normalization consistently prioritizes high-leverage perturbations and remains robust under bootstrap resampling and chemical similarity controls.

Together, these results demonstrate that proximity metrics are ill-suited for prioritization when treated inferentially, and that influence-based propagation provides a more stable framework for comparative network toxicology.
More broadly, this work clarifies how distinct network metrics should be interpreted and applied when reasoning about compound-associated risk in complex biological systems.

\noindent\textbf{Keywords:} network propagation, proximity metrics, drug-induced liver injury, polypharmacology risk assessment, random walk with restart, per-target influence normalization
