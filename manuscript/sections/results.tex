\section{Results}

\subsection{Network context}

Quercetin engages 62 targets in the liver-expressed largest connected component; Hyperforin engages 10 (Figure~\ref{fig:fig1}).
Shortest-path proximity to 82 DILI genes: Quercetin Z = $-$5.4 ($p < 10^{-6}$); Hyperforin Z = $-$2.6 ($p = 0.005$).
Negative Z-scores indicate closer-than-random proximity.
Target count and proximity are descriptive; neither constitutes a test of functional influence.

\subsection{Proximity does not predict influence}

Random walk with restart (RWR) quantifies influence propagation from compound targets to DILI genes (Figure~\ref{fig:fig2}).
Hyperforin: Z = +10.3 ($p < 10^{-16}$).
Quercetin: Z = +4.4 ($p < 10^{-6}$).
$n = 1{,}000$ degree-matched permutations.
Hyperforin exerts greater influence despite lower proximity.
This rank reversal is the primary inferential result.

\subsection{Expression weighting narrows the gap}

Expression-weighted influence (EWI) incorporates liver-specific gene expression into propagation (Figure~\ref{fig:fig3}).
The Z-score differential narrows: $\Delta$Z = +5.8 (RWR) $\rightarrow$ +3.5 (EWI).
Hyperforin loses 1.2 units; Quercetin gains 1.1 units.
The residual gap remains significant ($p < 10^{-8}$).
Expression weighting attenuates but does not reverse the ranking.

\subsection{Per-target efficiency}

Per-target normalized influence (PTNI = total influence / target count) normalizes for target set size (Figure~\ref{fig:fig4}).
Hyperforin: PTNI = 0.0114 (RWR), 0.0133 (EWI).
Quercetin: PTNI = 0.0005 (RWR), 0.0008 (EWI).
Ratio: 22-fold (RWR), 17-fold (EWI).
PTNI is an effect-size normalization; no separate permutation test was performed.

\subsection{Bootstrap sensitivity}

100 random 10-target subsets were sampled from Quercetin's 62-target pool and scored by RWR (Figure~\ref{fig:fig5}).
Hyperforin's observed influence exceeds the entire bootstrap distribution.
This excludes target count as a sufficient explanation for the observed ranking.
Bootstrap analysis is a robustness control, not additional evidence.

\subsection{Chemical similarity}

Maximum Tanimoto similarity to DILIrank reference drugs: Hyperforin = 0.18; Quercetin = 0.21 (Figure~\ref{fig:fig6}).
Both fall below the 0.4 threshold for structural analog detection \citep{Maggiora2014}.
This excludes structural resemblance to known hepatotoxins as an explanation for the network signal.
Chemical similarity is an orthogonal exclusion criterion, not a risk predictor.
